# PNPM Workspace Monorepo Boilerplate

This project is a boilerplate of a simple monorepo with pnpm workspace


## Features

- Monorepo with :
  - Frontend app with Next.js
  - Frontend app with Vite
  - ESLint configuration
  - Storybook
  - Utils functions package
